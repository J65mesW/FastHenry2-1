FastHenry2 Tutorial

Getting started with FastHenry2

This tutorial will show you step-by-step how to perform the basic inductance extraction tasks with FastHenry2.

In this tutorial, you will learn how to:

  • Load a FastHenry2 input file into FastModel
  • Understand basic FastHenry2 input file syntax
  • Launch FastHenry2 from FastModel
  • Understand FastHenry2 output
  • Specify a finer discretization
  • Define uniform ground planes
