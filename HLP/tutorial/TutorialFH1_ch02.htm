Load a FastHenry2 input file into FastModel

Launch FastModel

  1. If FastModel is not already running, double click the FastModel icon on your desktop or select 'FastModel' from your Start menu in Windows.
  2. Click on the I agree button to accept FastModel's license.

Load a FastHenry2 input file

  1. Select the FastHenry2 View (the view should be in foreground with the caption highlighted).
  2. From the File menu select File->Open FastHenry2 File or click on the icon on the toolbar.
  3. Browse to the software installation directory (e.g. C:\Program Files\FastFieldSolvers) and under FastHenry2\Samples\Input_files select the file square.inp, then click Open.

    A model of a square loop will appear in the FastModel View.
