FastHenry2Object.Quit

boolean FastHenry2Object.Quit()

Return Value

If true, FastHenry2 was able to quit. If false, FastHenry2 couldn't quit (possibly because there's a simulaton on going; try to stop it before quitting).

Parameters

None

Remarks

Quits FastHenry2.
