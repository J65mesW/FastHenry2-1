FastHenry2 Automation Example Programs

Altough Automation is a complex topic, using FastHenry2 via Automation is simple enough. To get you started, we have created some example scripts which can be used to learn how to control FastHenry2 and as a base to build more complex applications. In spite of their short length, these examples are quite comprehensive and the source code is well commented.

You can find the example files in a sub-directory in FastHenry2's installation directory.

FastHenryClient Sample Program

This is the most complete example, which shows the use of all FastHenry2 methods and properties. To compile this example, you must have MS Visual Basic 6.0 (or newer) installed.

fh_drive.xls Sample Program

This is a Visual Basic for Applications example. To run this example, you must have MS Excel 2000 (or newer) installed.

Remark: for the script to work, you must answer 'yes' at the prompt which asks you if you want to enable the macros, when you load the file into Excel.

Remark: the files coils1.inp, coils2.inp, coils3.inp are used by this script and must reside in the same directory of fh_drive.xls for the script to work.

fhdriv.vbs Sample Program

This is a Visual Basic Script example. To run this example, you only need one of the following Windows versions: Me, 2000, XP. The best way to launch this script is from a DOS shell, typing "cscript fhdriv.vbs", in the directory where fhdriv.vbs resides. You can also double-click on the file name, but in this case all the log messages will appear in separate pop-up message boxes.

Remark: the files coils1.inp, coils2.inp, coils3.inp are used by this script and must reside in the same directory of fhdriv.vbs for the script to work.
