FastHenry2Object.GetResistance

VARIANT FastHenry2Object.GetResistance()

Return Value

The method returns a 3-dimensional safe-array encapsulated in a variant data.

The first dimension identifies the matrix solved for a particular frequency. To know the frequency, you must query the frequency array via the GetFrequencies Method. The index of both arrays is the same, e.g. the second matrix in the resistance array is calculated at the frequency in the second position of the frequency array.

The second and third dimensions correspond to the row and column index respectively.

If the number of columns equals the number of rows of the array, the array contains the resistance matrix as calculated by FastHenry2, in Ohms.

If the number of columns of the array is not equal to the number of rows, the array contains the real part of the rectangular admittance matrix (this case happens when option '-x' is used running FastHenry2, see FastHenry user's manual).

Notes:

The array dimensions can be queried from the safe array itself, see the 'safe array' topic in the MS Automation documentation.

Parameters

None

Remarks

Gets the resistance or the real part of the admittance values calculated by FastHenry2, result of the simulation.
