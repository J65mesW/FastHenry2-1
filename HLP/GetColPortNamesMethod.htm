FastHenry2Object.GetColPortNames

VARIANT FastHenry2Object.GetColPortNames()

Return Value

The method returns a 1-dimensional safe-array encapsulated in a variant data. The array contains the names of the ports to which the resistance, inductance, and admittance matrices columns refer.

Notes:

The array dimensions can be queried from the safe array itself, see the 'safe array' topic in the MS Automation documentation.

Parameters

None

Remarks

Gets port names for the columns of the resistance, inductance, and admittance matrices.
