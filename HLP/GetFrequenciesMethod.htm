FastHenry2Object.GetFrequencies

VARIANT FastHenry2Object.GetFrequencies()

Return Value

The method returns a 1-dimensional safe-array encapsulated in a variant data. The array contains the frequencies at which the resistance, inductance or admittance matrices are calculated.

Parameters

None

Remarks

Gets the list of frequency points at which FastHenry2's results are calculated.
